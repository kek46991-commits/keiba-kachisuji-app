# 🏇 勝ち筋解析システム

**かずさん理論 × 物理統計** — パドック解析で期待値を最大化する競馬分析アプリ

## 機能

- **レース設定**: 全10競馬場対応、天候・馬場状態・距離の設定
- **パドック解析**: ストライド、弾み、毛艶、発汗、集中度などの指標入力
- **勝ち筋判定**: 物理補正 + コンディション補正 → 期待値算出
- **ボックス買い目計算機**: 三連単/三連複/馬連/馬単/ワイドの点数・投資額計算
- **ビジュアル分析**: Plotlyによるインタラクティブなグラフ表示

## セットアップ

```bash
pip install -r requirements.txt
```

## 起動

```bash
streamlit run app.py
```

## スコアリング

| 要素 | 説明 |
|------|------|
| 基礎点 | 60点（全馬共通） |
| ストライド×コーナーR | 後足の横振り幅とコースのR値の相関 |
| 直線距離補正 | 直線が長いほどストライドが活きる |
| 坂路適性 | 坂のあるコースでの弾み（バウンス）評価 |
| 低温×高弾み | 寒い日のぴょこぴょこは高評価 |
| ボロ後リラックス | パドックでのボロ後の歩様改善を検知 |
| 毛艶 | 体調の良さを反映 |
| 発汗 | 過度な発汗はマイナス評価 |
| 耳の動き | 集中度の指標 |
| 馬場悪化 | 重馬場ペナルティ（芝 > ダート） |

## 期待値の判定基準

| 期待値 | 判定 |
|--------|------|
| 3.0以上 | 🔥 激アツ・勝ち筋確定 |
| 2.0以上 | ⭐ 高期待値・要注目 |
| 1.5以上 | 👀 やや有望 |
| 1.0以上 | △ 普通 |
| 1.0未満 | ✗ 見送り推奨 |
