CORE FEATURES

3つのコア機能で「買うべき馬」を数値化する

📝

馬データ入力

出走馬の馬番・馬名・オッズを入力またはCSV貼り付けで一括インポート。出馬表をそのまま貼り付けるだけで即座に解析開始できます。

📊

解析結果

期待値・合計スコア・勝率を自動計算し、全馬をランキング表示。期待値が高い「激アツ馬」を自動ハイライトします。

🎰

買い目計算

ボックス買い目の点数・投資額を自動算出。馬券種別・頭数を選ぶだけで投資額が即座に計算されます。

SCORE SYSTEM

スコア = 基礎点 + 物理補正 + 状態補正で構成される

要素 内容 影響度
基礎点 オッズから逆算した市場評価（最大60点） 大
物理補正 馬場含水率・天候・距離・コーナー半径による補正 中
状態補正 馬場状態（良〜不良）による加算 小〜中
  • 合計スコアが高いほど「期待値あり・買い」判定
  • 期待値1.0以上が買い推奨ライン（控除率25%を考慮）
  • 🔥 妙味大・強い買い → ⭐ 期待値あり・買い → 🔄 損益分岐近し の3段階判定
  • 馬場・天候・距離を考慮した補正スコアで精度を向上
SETTINGS

開催場所・天候・馬場含水率まで設定して精度を高める

🏟️ 開催場所

阪神・京都・東京・中山・小倉・函館など主要競馬場を選択。コース特性に応じた補正が自動適用されます。

🌦️ 天候

晴れ・曇り・雨・雪を選択。天候ごとの馬場への影響を補正スコアに反映します。

💧 馬場含水率

良・稍重・重・不良の4段階から選択。含水率が高いほど先行有利の傾向を加味します。

📏 距離・コース

芝・ダート、距離（1000m〜3600m）を設定。距離適性とコーナー半径を補正に組み込みます。

HOW TO USE

出馬表を貼り付けて「解析実行」を押すだけ

  1. 出馬表を入力

    馬番・馬名・オッズを手入力、またはCSV形式で一括貼り付け。出馬表をそのままコピペするだけでOK。

  2. レース条件を設定

    開催場所・天候・馬場状態・距離・コースを選択。設定するほど補正精度が上がります。

  3. 解析実行

    「解析実行」ボタンを押すだけ。全馬の期待値・スコア・判定が瞬時に表示されます。

  4. 買い目を確認

    「買い目計算」タブで馬券種別・頭数を選択。投資額と点数が自動計算されます。
